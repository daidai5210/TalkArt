# 沟通规则

> **项目**: TalkArt
> **版本**: v0.1.0-mvp

---

## 沟通渠道

| 渠道 | 用途 |
|---|---|
| 飞书 P2P | 主 AIC ↔ 用户（需求确认、方案决策、验收） |
| 子 Agent 输出 | 子 Agent → 主 AIC（代码变更、测试结果、commit 信息） |

---

## 决策权限

| 决策类型 | 决策人 |
|---|---|
| 需求变更 | 用户 |
| 架构决策 | 主 AIC（记录在 `risk-and-blockers.md`） |
| 技术选型 | 主 AIC |
| 代码实现 | 子 Agent（主 AIC 审查） |
| 发布 | 用户确认 |
| 验收 | 用户 |
