# 阶段任务拆解：TalkArt build 阶段

> **版本**: v0.1.0-mvp
> **阶段**: build (Phase-04)

---

## 任务总览

```
Build 阶段拆为 5 个 Task Group (TG)：

TG-1: 项目脚手架 + SVG 渲染 (2 个子任务)
  ├── 1.1 项目初始化
  └── 1.2 SVG 渲染引擎

TG-2: 绘图工具函数库 (2 个子任务)
  ├── 2.1 基础图形工具 (6 个)
  └── 2.2 元素操作 + 画布工具 (9 个)

TG-3: AI Agent + BFF (2 个子任务)
  ├── 3.1 BFF API Route
  └── 3.2 AI Agent 对话管理

TG-4: 语音输入模块 (2 个子任务)
  ├── 4.1 语音管理器
  └── 4.2 唤醒词 + 结束语检测

TG-5: UI 集成 + 全链路联调 (2 个子任务)
  ├── 5.1 UI 组件集成
  └── 5.2 测试 + 文档
```

---

## 详细任务

### TG-1: 项目脚手架 + SVG 渲染

| ID | 任务 | 预估 | 依赖 | 验收标准 |
|---|---|---|---|---|
| 1.1 | 项目初始化 (Vite + React + TS + Tailwind + Zustand) | 0.5d | — | `npm run dev` 正常启动 |
| 1.2 | SVG 渲染引擎 (Canvas 组件 + ElementRenderer + 选中高亮 + CanvasStore) | 1.5d | 1.1 | 可手动添加 SVG 元素并渲染 |

**总预估**: 2 人天

### TG-2: 绘图工具函数库

| ID | 任务 | 预估 | 依赖 | 验收标准 |
|---|---|---|---|---|
| 2.1 | 基础图形工具 (drawRect/Circle/Ellipse/Line/Text/Triangle) + 语义→坐标映射 | 2d | 1.2 | 6 个函数单元测试通过 |
| 2.2 | 元素操作工具 (select/update/delete/move/scale/duplicate) + 画布工具 (clear/undo/export) | 2d | 2.1 | 9 个函数单元测试通过 + 撤销/重做测试 |

**总预估**: 4 人天

### TG-3: AI Agent + BFF

| ID | 任务 | 预估 | 依赖 | 验收标准 |
|---|---|---|---|---|
| 3.1 | Vercel API Route (`/api/llm`) + System Prompt + 15 个工具定义 | 1d | — | API Route 返回 LLM 确认/function_call |
| 3.2 | AI Agent 对话管理 (AgentStore + ConversationManager + ToolDispatcher + LLM Client) | 2d | 3.1 | Mock LLM 场景下，确认→执行→工具调用全链路 |

**总预估**: 3 人天

### TG-4: 语音输入模块

| ID | 任务 | 预估 | 依赖 | 验收标准 |
|---|---|---|---|---|
| 4.1 | 语音管理器 (麦克风权限 + Web Speech API 封装 + 转录文本展示) | 1.5d | 1.1 | 浏览器中说出中文 → 页面上显示识别文本 |
| 4.2 | 唤醒词 + 结束语检测 (WakeWordDetector + EndPhraseDetector) | 1d | 4.1 | "小智小智"触发监听 → "开始吧"触发执行 |

**总预估**: 2.5 人天

### TG-5: UI 集成 + 全链路联调 + 文档

| ID | 任务 | 预估 | 依赖 | 验收标准 |
|---|---|---|---|---|
| 5.1 | UI 组件集成 (全套组件 + 状态机联调) | 1.5d | 2.2, 3.2, 4.2 | 完整语音交互流程可跑通 |
| 5.2 | 测试 (单元测试补全 + E2E) + 设计文档 | 2d | 5.1 | 测试通过 + 指令能力清单完成 |

**总预估**: 3.5 人天
