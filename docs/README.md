## 项目简介

纯语音控制的 AI SVG 矢量绘图 Web 工具。用户通过语音指令，AI Agent（LLM Function Calling）自动选择和执行绘图工具函数，在网页中生成/操作 SVG 图形。不需要鼠标键盘，不需要学软件。
