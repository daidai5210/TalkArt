# TalkArt 纯语音演示脚本

## 启用演示模式

- 浏览器访问：`http://localhost:5173/?demo=1`
- 或在控制台执行：`localStorage.setItem('talkart-demo-mode','1'); location.reload()`

演示模式下隐藏：底部文字输入、工具栏、顶部导出按钮。麦克风按钮仍可用于开始/停止聆听。

## 演示流程（全程免键盘鼠标）

| 步骤 | 用户语音 | 预期行为 |
|:---:|---|---|
| 1 | 「小智小智」 | 唤醒，进入聆听 |
| 2 | 「画一个红色的圆在中间」 | AI 确认意图 |
| 3 | 「开始吧」 | 执行绘图，画布出现红圆 |
| 4 | （等待 TTS） | 播报确认语摘要 |
| 5 | 「撤销」 | 撤销上一步，TTS「已撤销」 |
| 6 | 「画三个蓝色圆排成一行」 | AI 确认 |
| 7 | 「可以了」 | `executeDrawingPlan` 批量绘制 |
| 8 | 「导出」 | 下载 SVG，TTS「已导出」 |

## 语音快捷指令（不经 LLM）

| 口令 | 动作 |
|---|---|
| 撤销 / 撤回 / 上一步 | `undo` |
| 导出 / 保存 / 下载 | 导出 SVG |

## 纠错词

| 口令 | 动作 |
|---|---|
| 不对 / 重新来 | 放弃当前确认，回到聆听 |
