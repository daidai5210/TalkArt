## 九、技术限制与已知问题

1. **浏览器兼容性**：Web Speech API 仅 Chrome/Edge 完整支持，Firefox/Safari 需文本兜底
2. **唤醒词检测**：客户端正则匹配，非专用唤醒词模型，噪音环境可能误触发
3. **空间推理**：LLM 空间推理依赖语义→坐标映射，极端复杂布局可能需要多次调整
4. **画布状态**：仅存浏览器内存（Zustand store），刷新页面丢失
5. **元素选择**：描述匹配基于关键词评分，多同类元素时可能选中非目标元素
6. **坐标精度**：语义位置为预设区域映射，无法精确到"红色圆的右边"等相对定位
7. **LLM 依赖**：核心对话和工具调用依赖外部 LLM API，离线无法使用
8. **语音识别**：ASR 准确率受环境噪声、口音影响，长句识别可能分段
