# 数据库设计：TalkArt v0.1.0-mvp

> **版本**: v0.1.0-mvp
> **状态**: ⚠️ 不适用

---

## 说明

TalkArt v0.1.0-mvp 是纯前端应用，**不使用数据库**。

所有状态（画布元素、对话历史、操作历史）仅存储在浏览器内存中（Zustand Store），页面刷新后丢失。

---

## 未来考虑

如果后续版本需要持久化（如保存作品、用户账户），可考虑：

| 方案 | 适用场景 |
|---|---|
| localStorage | 简单本地保存 |
| IndexedDB | 大量本地数据 |
| Vercel KV | 服务端轻量存储 |
| Supabase | 用户账户 + 云端存储 |
