## 二、技术风险

### R1: LLM 空间推理不精确

| 维度 | 内容 |
|---|---|
| **风险等级** | 🔴 高 |
| **概率** | 中（40-60%） |
| **影响** | 高 — 用户说"画在右边"，LLM 输出坐标不准确 |
| **触发条件** | 复杂空间描述、多元素相对位置 |
| **缓解措施** | 工具函数侧做语义→坐标映射；提供视觉反馈让用户纠正 |
| **残留风险** | 中 — 极端复杂布局可能需要用户多次调整 |
| **监控指标** | 用户"修改位置"类指令的频率 |

### R2: 用户不接受纯语音交互

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **概率** | 中（30-50%） |
| **影响** | 高 — 核心假设被推翻 |
| **触发条件** | 用户在公共场合/噪音环境使用；语音交互习惯未建立 |
| **缓解措施** | 文本输入兜底；MVP 阶段通过用户测试快速验证 |
| **残留风险** | 中 — 如果语音交互不被接受，产品需要 pivot |
| **监控指标** | 语音使用率 vs 文本兜底使用率 |

### R3: 端到端延迟过高

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **概率** | 中（30-50%） |
| **影响** | 中 — 用户体验下降，但可通过 UI 优化缓解 |
| **触发条件** | LLM API 响应慢（>3s）、网络延迟 |
| **缓解措施** | 流式响应、骨架动画、乐观更新、切换更快模型 |
| **残留风险** | 低 — 有多种优化手段 |
| **监控指标** | P50/P95 端到端延迟 |

### R4: Web Speech API 中文准确率不足

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **概率** | 中（30-50%） |
| **影响** | 中 — 识别错误导致 LLM 收到错误输入 |
| **触发条件** | 口音、噪音、专业术语 |
| **缓解措施** | 显示识别文本让用户确认；文本兜底修正；生产切 Whisper |
| **残留风险** | 低 — 可升级 ASR 引擎 |
| **监控指标** | ASR 识别准确率、用户手动修正率 |

### R5: 复杂指令拆解失败

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟢 低（MVP 阶段） |
| **概率** | 中 |
| **影响** | 低 — MVP 只支持单步指令，此风险在 P1 阶段 |
| **缓解措施** | MVP 不覆盖；P1 阶段用 Chain-of-Thought 提示词 |
| **残留风险** | 低 |

### R6: 长对话上下文溢出

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟢 低 |
| **概率** | 低（<20%） |
| **影响** | 中 — 长对话后 LLM 丢失画布上下文 |
| **缓解措施** | 画布摘要注入（而非全量历史）；滑动窗口 |
| **残留风险** | 低 |

## 三、市场风险

### R7: 大厂进入

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **概率** | 低（<20%，12 个月内） |
| **影响** | 高 — Figma/Canva 加入语音控制将直接竞争 |
| **缓解措施** | 先发优势 + 专注语音原生体验（大厂只会加"语音按钮"） |
| **残留风险** | 中 — 如果大厂全力投入，小团队难以竞争 |
| **监控指标** | Figma/Canva/Adobe 产品更新动态 |

### R8: LLM API 成本不可控

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟢 低 |
| **概率** | 低 |
| **影响** | 中 — 成本随用户量线性增长 |
| **缓解措施** | 开源模型（DeepSeek/Qwen）成本极低；缓存策略 |
| **残留风险** | 低 |

## 四、合规与安全风险

### R9: 用户隐私（语音数据）

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **说明** | 语音数据可能包含敏感信息 |
| **缓解措施** | MVP 使用浏览器本地 ASR（Web Speech API），语音不出浏览器；生产环境明确隐私政策 |
| **合规要求** | GDPR（如服务欧洲用户）、中国个人信息保护法 |

### R10: 内容安全（生成内容）

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟢 低 |
| **说明** | 本产品生成 SVG 图形而非文本/图像内容，不当内容风险较低 |
| **缓解措施** | 文本工具（drawText）做基本敏感词过滤 |

### R11: API Key 安全

| 维度 | 内容 |
|---|---|
| **风险等级** | 🟡 中 |
| **说明** | LLM API key 不能暴露在前端代码中 |
| **缓解措施** | BFF（Vercel API Route）代理，API key 仅存服务端环境变量 |
