## 一、核心价值主张

### 一句话版本

> **说话就能画矢量图。** 不需要学软件，不需要鼠标键盘，说出你想要的就出现在画布上。

### 详细版本

纯语音控制的 AI SVG 绘图工具让任何人都能通过自然语言创建和编辑矢量图形。AI Agent 理解你的创作意图，调用绘图工具执行精确的 SVG 操作。不是 AI 生成不可控的像素图，而是你指挥 AI 精确绘制可编辑的矢量图。

| 维度 | 传统 SVG 编辑器 | AI 文生图工具 | **我们** |

| **输出类型** | 可编辑矢量图 | 不可编辑栅格图 | **可编辑矢量图** |

| **无障碍** | ❌ | 部分（文本输入） | **✅ 天然无障碍** |

## 三、创新点

### 3.1 架构创新：Agentic Drawing Pipeline

```
传统绘图工具:
  用户 → 找到工具 → 鼠标操作 → 渲染
  瓶颈：用户必须知道"用什么工具"和"怎么操作"

AI 文生图工具:
  用户 → 文本提示词 → AI 生成 → 栅格图输出
  瓶颈：不可精确控制，不可增量修改

我们的 Agentic Pipeline:
  用户 → 语音 → ASR → LLM(Function Calling) → 绘图工具 → SVG 渲染
  优势：用户只需要"说什么"，AI 负责"用什么工具"和"怎么执行"
```

### 3.2 交互创新：语义空间理解

LLM 将自然语言中的空间描述翻译为精确坐标：

| 用户说 | LLM 理解并映射到 |
|---|---|
| "在画布中间画一个圆" | `drawCircle({cx: canvasWidth/2, cy: canvasHeight/2, r: 100})` |
| "右边再画一个" | `drawCircle({cx: prevObj.x + prevObj.width + gap + r, cy: prevObj.cy, r: 100})` |
| "这个改成红色" | `updateElement({id: activeElement, fill: "#FF0000"})` |
| "大一点" | `scaleElement({id: activeElement, scale: 1.2})` |

### 3.3 体验创新：增量式语音修改

这是**最核心的创新**——与传统 AI 画图"不满意就重新生成"完全不同：

```
用户: "画一个蓝色矩形"
画布: [蓝色矩形出现]
用户: "中间加一段文字'Hello'"
画布: [矩形中间出现文字]
用户: "把矩形改成圆角，颜色深一点"
画布: [矩形变圆角，颜色加深]
```

每个操作都是对画布上**现有元素**的增量修改，而非全量重新生成。这需要在对话上下文中维护画布状态和元素引用。

### 3.4 产品创新：零学习曲线 + 高天花板

- **零学习曲线**：说出第一句话就能开始创作
- **高天花板**：随着工具函数增多，复杂度可无限扩展（图层、路径、渐变、动画）
