# API/数据/权限评审：TalkArt v0.1.0-mvp

> **版本**: v0.1.0-mvp

---

## 一、评审结论

| 维度 | 评审结果 | 说明 |
|---|---|---|
| API 安全 | ✅ 通过 | API Key 仅存 Vercel 环境变量，BFF 代理 |
| 数据安全 | ✅ 通过 | 无数据库，无用户数据存储 |
| 权限 | ⚠️ 不适用 | MVP 无登录/无用户管理 |

---

## 二、API 安全

| 检查项 | 状态 | 说明 |
|---|---|---|
| API Key 暴露 | ✅ 不暴露 | 仅存 `VERCEL_ENV`，前端不可见 |
| CORS | ✅ 同源 | BFF 与前端同域部署 |
| 速率限制 | ⚠️ 待加 | MVP 可后续加 Vercel Rate Limiting |
| 输入验证 | ✅ | BFF 层校验 messages/tools 格式 |

---

## 三、数据安全

| 检查项 | 状态 | 说明 |
|---|---|---|
| 用户数据存储 | ⚠️ 不存在 | 状态仅存浏览器内存 |
| 语音数据 | ✅ | Web Speech API 浏览器本地处理 |
| LLM 对话数据 | ⚠️ 传输中 | 经 BFF 转发至 LLM API，不存储 |

---

## 四、权限

MVP 无用户系统，无权限模型。后续如需添加：
- 公开画布（只读）
- 私有画布（需登录）
- 协作编辑（需权限控制）
