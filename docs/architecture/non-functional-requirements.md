# 非功能需求：TalkArt v0.1.0-mvp

> **版本**: v0.1.0-mvp

---

## 一、性能

| 指标 | 目标 | 测量 |
|---|---|---|
| 首屏加载 | <2s | Lighthouse |
| 唤醒词响应 | <500ms | Performance API |
| ASR 延迟 | <200ms | Performance API |
| LLM 确认对话 (P50) | <2s | API 耗时 |
| LLM 确认对话 (P95) | <5s | API 耗时 |
| 工具执行 | <10ms | 函数耗时 |
| SVG 渲染帧率 | >55fps | rAF |
| 导出 800×600 PNG | <500ms | 函数耗时 |

---

## 二、可用性

| 指标 | 目标 |
|---|---|
| 工具选择准确率 | >90%（20 个标准指令） |
| 唤醒词识别率 | >95%（安静环境） |
| ASR 识别率 | >85%（标准普通话） |
| 用户首次任务完成率 | >70% |

## 三、兼容性

| 平台 | 支持 |
|---|---|
| Chrome 100+ | ✅ 完全支持（Web Speech API 最佳） |
| Edge 100+ | ✅ 完全支持 |
| Firefox 120+ | ⚠️ Web Speech API 不支持，需文本兜底 |
| Safari 16+ | ⚠️ Web Speech API 部分支持 |
| 移动端 | ⚠️ 不强制（P2 PWA） |

---

## 四、安全

| 要求 | 实现 |
|---|---|
| API Key 保护 | BFF 代理，环境变量 |
| 语音数据 | 浏览器本地处理（Web Speech API） |
| XSS | React 默认转义 + CSP |
| HTTPS | Vercel 默认启用 |
