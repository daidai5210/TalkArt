## 2. 功能验收标准

### 2.1 语音交互模块

| ID | 功能 | 验收标准 | 测试方式 |
|---|---|---|---|
| V-01 | 唤醒词检测 | 说"嘿小智"/"小智小智"/含"小zhi"同音词时，系统进入监听模式 | 手动测试 |
| V-02 | 语音采集 | 点击页面后浏览器请求麦克风权限，授权后可录音 | 手动测试 |
| V-03 | ASR 转文字 | 说中文后，页面显示识别文本，准确率 >85% | 手动测试 |
| V-04 | 识别文本展示 | 页面上方实时显示"你说的：xxx" | 目视检查 |
| V-05 | 结束语检测 | 说"开始吧"/"可以了"后，系统执行绘图 | 手动测试 |
| V-06 | 纠错支持 | 说"不对"后，系统允许重新描述 | 手动测试 |

### 2.2 AI Agent 模块

| ID | 功能 | 验收标准 | 测试方式 |
|---|---|---|---|
| A-01 | 意图分析 | LLM 正确理解绘图意图，输出结构化确认 | 手动 20 个指令测试 |
| A-02 | 多轮确认 | LLM 反问确认需求，对话自然流畅 | 手动测试 |
| A-03 | Function Calling | 确认后调用正确的工具函数，参数合理 | 手动测试 + 单元测试 |
| A-04 | 画布上下文 | 修改指令能引用现有元素（"那个圆"） | 手动测试 |
| A-05 | 模糊指令 | "大一点"→选中元素放大 1.2x | 手动测试 |
| A-06 | 错误恢复 | 工具失败时提示用户，不崩溃 | 手动测试 |

### 2.3 绘图工具函数

| ID | 验收标准 |
|---|---|
| T-01~T-06 | 每个工具函数接收合法参数后，SVG 元素正确渲染 |
| T-07~T-12 | 选中、修改、删除、移动、缩放、复制操作正确执行 |
| T-13 | clearCanvas 清空所有元素 |
| T-14 | undoAction 正确回退到上一步状态 |
| T-15 | exportImage 输出合法 SVG/PNG 文件 |

### 2.4 导出模块

| ID | 验收标准 |
|---|---|
| E-01 | 导出 SVG 文件内容合法，浏览器可打开 |
| E-02 | 导出 PNG 文件为有效图片，尺寸与画布一致 |

## 3. 性能验收标准

| 指标 | 阈值 | 测量方式 |
|---|---|---|
| 唤醒词响应延迟 | <500ms | 手动计时 |
| ASR 转文字延迟 | <200ms | Performance API |
| LLM 确认对话延迟 | <2s（P50）/ <5s（P95） | API 耗时打点 |
| 工具执行延迟 | <10ms | 函数耗时打点 |
| SVG 渲染延迟 | <16ms（60fps） | requestAnimationFrame |
| 页面首屏加载 | <2s | Lighthouse |

## 4. 质量验收标准

| 指标 | 阈值 |
|---|---|
| 工具选择准确率 | >90%（20 个标准指令） |
| 用户任务完成率 | >70%（3 个测试任务） |
| JS 异常崩溃 | 0（核心流程） |
| 单元测试覆盖率 | >80%（工具函数 + 状态管理） |
