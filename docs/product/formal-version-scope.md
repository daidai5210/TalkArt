# 正式版范围草案：TalkArt v0.1.0-mvp

> **版本**: v0.1.0-mvp
> **路线**: A-L（轻量 MVP）

---

## 版本目标

发布 TalkArt 的最小可用版本，验证核心假设：纯语音控制 + AI Agent + SVG 矢量绘图的可行性。

---

## 版本范围

### 核心用户流程（端到端）

```
打开页面 → 唤醒("小智小智") → 语音描述需求
  → AI 多轮确认 → 用户确认("开始吧")
  → 工具执行 → SVG 渲染 → 导出
```

### 包含的功能模块

| 模块 | 功能点数 |
|---|---|
| 语音交互 | 6（唤醒、采集、ASR、展示、结束、纠错） |
| AI Agent | 6（意图、确认、FC、上下文、消歧、恢复） |
| 绘图工具 | 15（6 基础 + 6 操作 + 3 画布） |
| SVG 渲染 | 4（渲染、选中、预览、状态） |
| 导出 | 2（SVG + PNG） |
| 设计文档 | 1（指令能力清单） |
