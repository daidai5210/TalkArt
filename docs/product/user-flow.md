# 用户流程：TalkArt

> **版本**: v0.1.0-mvp

---

## 主流程

```
[打开页面]
    │
    ▼
[画布就绪，显示麦克风按钮]
    │
    ▼
[用户说 "小智小智"]
    │
    ▼
[系统: "我在听" 🎤]
    │
    ▼
[用户说绘图需求]
    │
    ▼
[ASR 转文字，页面上显示 "你说的：xxx"]
    │
    ▼
[AI Agent 分析意图]
    │
    ├── 意图清晰 ──→ [反问确认: "你想画xxx，对吗？"]
    │                    │
    │                    ├── "对" / "开始吧" ──→ [执行绘图] ──→ [SVG 渲染]
    │                    │
    │                    └── "不对" ──→ [返回监听，重新描述]
    │
    └── 意图模糊 ──→ [追问: "你想画什么类型？"]
                        │
                        └── 用户回答 ──→ [确认] ──→ [执行]
```

## 导出流程

```
[画布上有图形]
    │
    ▼
[用户说 "小智小智，导出为PNG"]
    │
    ▼
[AI: "你想导出为PNG格式，文件名是 drawing.png，开始吧？"]
    │
    ▼
[用户: "可以了"]
    │
    ▼
[浏览器下载 PNG]
```

---

## 修改流程

```
[画布上有图形]
    │
    ▼
[用户: "小智小智，把那个圆改成红色"]
    │
    ▼
[AI: "你想把画布上的圆形改成红色，对吗？"]
    │
    ▼
[用户: "对，开始吧"]
    │
    ▼
[系统: selectElement → updateElement(fill: "#FF0000")]
    │
    ▼
[圆形变红色，选中高亮]
```
