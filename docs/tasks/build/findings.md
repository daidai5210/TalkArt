## 发现记录

### F-001: TypeScript 类型断言需补强
- **发现时间**: 2026-06-12
- **模块**: drawing-tools/basic-shapes.test.ts
- **描述**: `drawLine` 测试中 `result.element!.props.x1` 返回 `unknown` 类型，直接传给 `toBeLessThan()` 导致 TS2345 编译错误
- **修复**: 添加 `as number` 类型断言
- **影响**: 仅影响编译，无运行时行为变化

### F-002: 元素描述匹配为启发式评分
- **发现时间**: 2026-06-12
- **模块**: drawing-tools/element-ops.ts
- **描述**: `findElementByDescription` 使用关键词评分匹配（类型2分+颜色1分），当画布上有多个同类元素时可能选中非目标元素
- **建议**: 后续版本可加入序号（"第二个圆"）或位置描述（"左边的圆"）增强匹配精度

### F-003: 画布状态无持久化
- **发现时间**: 2026-06-12
- **模块**: store
- **描述**: Zustand store 仅存内存，页面刷新后所有元素丢失
- **建议**: 后续版本可加 localStorage 持久化中间件
